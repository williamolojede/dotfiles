Installation

    git clone git@github.com:williamolojede/dotfiles.git

Create symlinks:

    ln -s ~/dotfiles/vimrc ~/.vimrc
    ln -s ~/dotfiles/gvimrc ~/.gvimrc
    ln -s ~/dotfiles/vim ~/.vim
    ln -s ~/dotfiles/gitconfig ~/.gitconfig
    ln -s ~/dotfiles/zshrc ~/.zshrc
